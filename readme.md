## Godab web source code

Código de la web Godab, usando Laravel Framework y varias librerías.
Cuenta con panel de administración.

### Instrucciones

Una vez clonado el repositorio, hacer los siguientes comandos por consola:

    -composer update
    -bower update
    -npm update
    -artisan migration
    -gulp
